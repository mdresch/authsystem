

# My Authentication System

Welcome to the documentation for My Authentication System!  This project provides a secure and user-friendly authentication solution built with Next.js, Supabase, and shadcn/ui.


  This documentation is a work in progress.


## Key Features

- User registration with email and password
- Secure login with password hashing
- Password reset functionality
- Simulated email verification (see [Getting Started](/dino/docs/getting-started) for details)
- Protected routes
- ... (list other features) ...

## Technologies Used

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- shadcn/ui Components
- Zod for form validation
- Lucide Icons
- Supabase
